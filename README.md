# QuizApp
Application console Python permettant une interaction entre un utilisateur à travers des questions sous plusieurs formats dans un domaine spécifique

**Objectif :** Concevoir une application de Quiz dynamique et pédagogique.

Les attendus du projet :

Structure par catégories : Les questions ne doivent pas être en vrac. Elles doivent être classées par thématiques (ex: Tech, Histoire, Langues, etc.).

Variété des supports/questions : L'application ne doit pas se limiter à un seul type de question. Elle doit intégrer au minimum des Questions à Choix Unique (QCU) et des Questions à Choix Multiples (QCM).

Le double système de Feedback :

Individuel : Après chaque question, l'utilisateur doit savoir s'il a bon ou faux, voir la correction et une explication.

Global : À la fin du quiz, un bilan complet doit être affiché (score, appréciation).

Votre mission : En tant que concepteurs, vous devez formaliser ce besoin en rédigeant le document de référence : le Cahier des Charges.
